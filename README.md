# Silicon Defense

An interview preparation platform that generates personalized study plans and practice questions using Google Gemini.

## Features

- Interview type selection (SWE, Product Management, Behavioral, General Career)
- Multi-phase study plans tailored to your timeframe and target companies
- Daily task scheduling with completion tracking
- Practice questions with difficulty levels and tips

## Tech Stack

- Frontend - React, TypeScript, Vite, Tailwind CSS
- Backend - Spring Boot, Spring Security
- Database - MongoDB 
- AI - Google Gemini API
